# Code of Conduct

This project welcomes contributions from everyone. To keep it a
pleasant place to collaborate:

- Be kind and respectful, and assume good faith.
- Disagreement is fine — critique ideas and code, not people.
- Harassment, personal attacks, and discriminatory language are not
  welcome, in any project space (issues, pull requests, reviews).
- Respect others' privacy; don't share personal information without
  permission.

The maintainer may edit or remove content and restrict participation
when these expectations are not met. Concerns can be reported
privately to na0@datanest.jp.
